# 🚀 Reports Zone

A modern, highly responsive, and robust web application designed for reporting and managing problems or issues seamlessly. Built with the power of Next.js App Router, Tailwind CSS, and Better-Auth.

---

## ✨ Features

- **🔒 Advanced Authentication:** Secure and session-based login/register powered by `Better-Auth`.
- **💻 Responsive Navigation:** Modern sticky header with an interactive mobile menu and smooth user profile dropdown.
- **🎨 Premium UI/UX:** Styled using `Tailwind CSS`, ambient background blur glow effects, and modern gradients.
- **🚦 Smooth UX Utilities:** Instant user action feedback via `React-Toastify` and modern iconography with `Gravity UI Icons`.
- **🧩 Custom Error Handling:** Clean and user-friendly custom `404 Not Found` page with a retro/minimalist stamp-style layout.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Authentication:** [Better-Auth](https://www.better-auth.com/)
- **Icons:** [Gravity UI Icons](https://gravity-ui.com/icons)
- **Notifications:** [React-Toastify](https://fkhadra.github.io/react-toastify/)
- **Fonts:** IBM Plex Mono & Inter
